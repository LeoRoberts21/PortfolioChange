DOUGH MOTHER

HTML CSS JavaScript

Project Purpose

I designed and coded a website for DoughMother, a local bakery. The goal of this was to improve my front-end skills, more specifically practising to code responsive sites. I decided that DoughMother would benefit from a website and could be reaching more customers if they had an online presence like their competition, some of whom have over 10k followers on Instagram.

Thought Process

The main goal of this project was to make a website that felt essential to the client with features that would enhance their customer's experience. I achieved this by using Figma, HTML and SCSS to design and code a website, responsive to various screen sizes. The site features information, pictures, recipes, and a mailing list. I also wanted to practise my JavaScript skills so added a working cart that lets the user add items to make online orders.
